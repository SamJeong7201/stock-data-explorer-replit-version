# OmniAlpha Labs 📈

**AI-powered global stock analysis platform** that aggregates worldwide market data, translates financial news, and generates both quick insights and deep AI-driven reports — all in one place.

🌐 **Live Demo**: [omnialpha.vercel.app](https://omnialpha.vercel.app)

---

## ✨ Features

- **Real-time Global Stock Data** — Fetches live prices, charts, and market stats via Yahoo Finance & Naver Finance APIs (US + Korean markets)
- **AI News Summaries** — Automatically ranks news by relevance and generates "Why it matters" summaries using GPT-4o-mini
- **Multilingual Support** — Full EN / KO / ZH translation of news headlines and summaries powered by OpenAI
- **AI Deep Reports** — On-demand AI-generated investment reports per ticker
- **Live Market Clock** — Shows real-time local market hours and open/closed status for each stock's exchange *(signature feature 🕐)*
- **Smart Watchlist** — Save and track your favorite tickers with portfolio-level insights
- **Sentiment Analysis** — Automatic positive / neutral / negative classification of every news headline

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, Vite, TypeScript, Tailwind CSS |
| Backend | Express.js, Node.js, TypeScript |
| AI | OpenAI GPT-4o-mini |
| Data | Yahoo Finance API, Naver Finance API |
| Deployment | Vercel (frontend) + Render (API server) |
| Package Manager | pnpm (monorepo / workspace) |

## ✨ 주요 기능

- **실시간 글로벌 주식 데이터** — Yahoo Finance & Naver Finance API를 통해 미국/한국 주식 실시간 가격, 차트, 지표 제공
- **AI 뉴스 요약** — GPT-4o-mini로 "왜 중요한가" 요약 자동 생성
- **다국어 지원** — EN / KO / ZH 뉴스 번역
- **AI 심층 리포트** — 종목별 AI 투자 리포트 즉시 생성
- **실시간 장 운영 현황** — 거래소별 현지 시간 및 장 개폐 여부 실시간 표시 *(시그니처 기능 🕐)*
- **스마트 관심종목** — 관심 종목 저장 및 포트폴리오 인사이트
- **감성 분석** — 뉴스 헤드라인 긍정/중립/부정 자동 분류
