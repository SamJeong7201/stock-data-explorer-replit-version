## Upgrade the AI Deep Report structure

Please redesign the premium report so it feels more like a real paid product.

Suggested structure:

1. Executive Summary

   * concise top-level read on the stock
   * 2–4 sentence professional overview

2. Market Positioning

   * what the market seems to be pricing in
   * current posture / pressure / sentiment

3. Key Drivers

   * the main forces moving the stock right now
   * company-specific + macro/sector context

4. Risk Assessment

   * near-term risks
   * what could invalidate the current thesis

5. Institutional / Flow Context

   * whether the move looks reactive, trend-following, hedging-driven, or conviction-driven
   * phrase carefully if data is inferred rather than directly known

6. Decision Context

   * what this setup means for:

     * short-term traders
     * swing traders
     * longer-term holders

7. What to Watch Next

   * 3 to 5 concrete things to monitor next

8. Confidence / Caveat Footer

   * clearly premium and polished, but not pretending certainty
   * should build trust, not overpromise
